
 🎥 Video NFT DEX – Decentralized Exchange for Video NFTs 🚀  

![Video NFT DEX](https://images.lumacdn.com/cdn-cgi/image/format=auto,fit=cover,dpr=2,quality=75,width=800,height=521.5584415584416/editor-images/vs/1b6718e2-66dc-4fce-84f7-18c5b5cf4d49.jpg)  

📌 Video NFT DEX is a decentralized exchange (DEX) that allows users to mint, buy, sell, and trade video NFTs securely and transparently on the blockchain. This project leverages smart contracts, IPFS, and Web3 technology to build a truly decentralized video NFT ecosystem.  

💡 Why Video NFT DEX?  
- Built for Video NFTs – A dedicated ecosystem for video content.  
- Copyright Protection – Ensure creators receive royalties on every trade.  
- Decentralized Trading – Peer-to-peer transactions with no intermediaries.  
- Secure Storage with IPFS – Immutable video and metadata storage.  

---

 🌟 Key Features  
✅ Mint Video NFTs – Easily convert videos into NFTs.  
✅ Decentralized Trading – Buy/sell Video NFTs on-chain using smart contracts.  
✅ Web3 Wallet Integration – Connect effortlessly with MetaMask, WalletConnect.  
✅ Transparent & Secure – All transactions are recorded on the blockchain.  
✅ IPFS & Thirdweb Storage – Decentralized and tamper-proof data storage.  
✅ Multi-chain Compatibility – Supports Ethereum, Polygon, Mammoth Blockchain.  

---

 🏗 System Architecture  
![Architecture](https://developers.moralis.com/wp-content/uploads/2023/10/Architecture-components-of-an-NFT-marketplace-and-what-developers-need-to-interact-with-1024x435.jpeg)

 1️⃣ Frontend (React + Thirdweb)  
- Next.js + React.js – Seamless UI with optimized performance.  
- Thirdweb SDK – Easy NFT minting, trading, and blockchain integration.  
- Ethers.js / Web3.js – Smart contract interaction.  
- Tailwind CSS – Lightweight and beautiful UI.  

 2️⃣ Backend (Node.js + MongoDB)  
- Express.js – Handles backend API.  
- MongoDB + Mongoose – Stores NFT metadata.  
- IPFS / Pinata / Thirdweb Storage – Decentralized video storage.  

 3️⃣ Smart Contract (Solidity + Hardhat)  
- Solidity – Smart contracts following ERC-721 / ERC-1155 standards.  
- Hardhat / Truffle – Deployment & testing on testnet/mainnet.  
- OpenZeppelin – Secure smart contracts.

 🚀 Getting Started  

 🔧 1. Clone the Project  
```bash
git clone https://github.com/dinhvaren/Video_NFT_DEX.git
cd Video_NFT_DEX
npm install
```

 📜 2. Configure Environment Variables  
Create a `.env` file and add the following information:  
```env
NEXT_PUBLIC_INFURA_ID=your_infura_id
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your_project_id
NEXT_PUBLIC_THIRDWEB_CLIENT_ID=your_thirdweb_client_id
MONGODB_URI=your_mongodb_connection_string
IPFS_API_KEY=your_ipfs_key
```

 ▶️ 3. Run the Project  
 Start Backend
```bash
cd my_app
npm run server
```
 Start Frontend
```bash
cd my_app
npm run build
npm start
```
Then open your browser and visit http://localhost:3000 to experience it!  

---

 📜 Deploying Smart Contracts  
To deploy smart contracts on a blockchain, run:  
```bash
npx hardhat run scripts/deploy.js --network <network>
```
Replace `<network>` with Ethereum, Polygon, Mammoth, or any supported blockchain.
